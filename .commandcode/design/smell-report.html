// PRIORITY ISSUES
P0

Generic color identity

The default theme uses blue (#2563eb) and the refresh uses teal (#0f8c9a). Both are standard SaaS/tech primaries. A GPA calculator for Australian high school students has no reason to look like a dashboard tool.

FIX: /design recolor. Pick a hue tied to the academic context, not the tech category. Consider warm or neutral tones that differentiate from typical edtech.
P0

Subject grid has no hierarchy

Core subjects (English, Math, Science at 1.0 weight) appear identical to electives (Drama, Music at 0.3 weight) in a uniform checkbox grid. The visual weight of a subject should reflect its academic weight.

FIX: /design relayout. Group core and elective subjects separately. Give core subjects visual prominence. Make the weight difference visible, not just a label.
P1

Icon toppers on every card

Every section title is prefixed with a Lucide icon (GraduationCap, BookOpen, Calculator, Trophy, Target, Palette). Icons serve no functional purpose and fill template slots.

FIX: /design deslop. Remove decorative icons from card titles. Keep icons only where they convey state or action (navigation, buttons).
P1

Center-stack composition

The selection screen centers everything: title, subtitle, grid, CTA. The target GPA screen centers a card in the viewport. No scan path, no composition decision.

FIX: /design relayout. Left-align the selection form for natural scan path. Treat the target input as a form step, not a marketing hero.
P2

GPA displayed as stat monument

The GPA number is 56-64px centered in a card with "out of 15.00" below. No context about which subjects contributed, what is missing, or how it compares to the target.

FIX: /design relayout. Pair the GPA number with a contribution breakdown. Show subject weights, entered grades, and missing data alongside the result.
P2

Unearned blur effects

backdrop-filter: blur(18px) on headers and cards. The "liquid-glass" name promises a material language the interface does not deliver. Blur without depth hierarchy is decoration.

FIX: /design surface. Either commit to a glass material system with clear depth planes, or remove the blur and use solid surfaces with border hierarchy.
